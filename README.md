# FindTimezone
## Setup
Download `timezones-now.geojson.zip` from:
https://github.com/evansiroky/timezone-boundary-builder/releases
Extract and place it in the project root directory.
## How to Use
`python3 main.py "Place of Interest"`,
<br>
`python3 main.py Place`,
<br>
or
```
>>> python3 main.py
Enter a place name: "Place of Interest"
```
